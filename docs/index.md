# FWXVI Strategy

**Midnight Sun Team #24 -- MSXVI Strategy Simulation & Optimization**

This project provides the simulation and race strategy tools for the Midnight Sun solar car.

## Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Run the interactive menu
cd simulation/
python main.py
```
